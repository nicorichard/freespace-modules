# freespace-modules

A collection of cleanup modules for [freespace](https://github.com/nicovince/freespace).

## Modules

| Module | Description |
|--------|-------------|
| `docker` | Docker Desktop disk images and build cache |
| `general-caches` | macOS and XDG application caches |
| `homebrew-cache` | Homebrew cached downloads and bottles |
| `npm-cache` | npm, Yarn, and pnpm package caches |
| `xcode-derived-data` | Xcode build artifacts and derived data |

## Installation

Install all modules:

```sh
freespace module install github:<user>/freespace-modules
```

Install a single module:

```sh
freespace module install github:<user>/freespace-modules#docker
```
